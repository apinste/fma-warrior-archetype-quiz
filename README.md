# Discover Your FMA Warrior Archetype  
A personality-style quiz designed for practitioners of Filipino Martial Arts (FMA) to explore their natural fighting tendencies through five traditional warrior archetypes: Tigre, Agila, Alon, Apoy, and Sawa.

This project powers the interactive quiz hosted on TrainPekiti.com, created for students and practitioners of Pekiti Tirsia Kali and other FMA systems in the DC/MD/VA region.

## About the Quiz
The FMA Warrior Archetype Quiz helps users identify their dominant warrior archetype based on mindset, movement, and combat preferences. The quiz is designed to be fun and accessible to all levels of FMA practitioners.

## Features
- Ten-question interactive quiz  
- Dynamic scoring and archetype assignment  
- Descriptions for all five warrior archetypes  
- Clean, mobile-friendly layout  
- Fully client-side (HTML/CSS/JavaScript)

## Technologies Used
- HTML5  
- CSS3  
- Vanilla JavaScript  

No external libraries or frameworks are required.
